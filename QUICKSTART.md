# 快速开始指南 🚀

## 📋 前置准备

确保你的系统已安装：
- ✅ Node.js (v16或更高版本)
- ✅ npm (v9或更高版本)

检查版本：
```bash
node --version
npm --version
```

## 🎯 3步启动项目

### 第1步：安装依赖
```bash
cd grade3-learning-assistant
npm install
```

### 第2步：启动开发服务器
```bash
npm run dev
```

### 第3步：打开浏览器
访问 http://localhost:3000

## 🔧 常见问题

### Q: 需要注册账号吗？
A: 不需要！只需输入名字即可开始。

### Q: 数据会丢失吗？
A: 数据保存在浏览器本地，清除缓存会丢失。

### Q: 支持离线使用吗？
A: 支持！首次加载后可离线使用。

### Q: 如何重置进度？
A: 清除浏览器缓存或使用隐身模式。

## 🚀 部署上线

### 一键部署（推荐）
```bash
# 运行部署脚本
./deploy.sh

# 然后选择平台：
# 1) Vercel - 最简单
# 2) Netlify - 功能丰富
# 3) GitHub Pages - 开源友好
# 4) Docker - 自托管
```

### 手动部署
```bash
# 1. 构建项目
npm run build

# 2. 上传 dist/ 目录到：
#    - Vercel: 拖拽 dist 到控制台
#    - Netlify: 拖拽 dist 到控制台
#    - 云服务器: scp -r dist/* user@server:/var/www/html
```

### 访问地址
- **本地开发**: http://localhost:3000
- **Docker部署**: http://localhost:8080
- **云部署**: 根据平台提供的域名
